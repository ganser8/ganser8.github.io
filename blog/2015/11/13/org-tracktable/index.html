Использование org-tracktable

Пакет org-tracktable предназначен для подсчёта количества написанных слов в org-документе, с занесением статистики в таблицу.

Установка

Поскольку на данный момент пакет недоступен в melpa, автор рекомендует клонировать репозиторий с github и прописать в init-файле следующее: 

(add-to-list 'load-path "~/.emacs.d/lisp/org-tracktable")
(load "org-tracktable.el")

Использование

После загрузки пакета, становятся доступны следующие функции:

org-tt-insert-table
вставляет таблицу
org-tt-write
добавляет в таблицу запись с текущим подсчётом слов. Если запись на текущий день уже существует, эта запись будет обновлена.
org-tt-status
покажет сообщение с общим количеством слов в буфере или в выделенной области.

После вставки таблица выглядит следующим образом: 

#+NAME: tracktable
|---+---------------+-----+-----+-------+---------|
| ! | date          | beg | end | total | comment |
|---+---------------+-----+-----+-------+---------|
|   | initial count |   0 |   0 |     0 |         |
|---+---------------+-----+-----+-------+---------|
#+TBLFM: @2$2=initial count::$2='(org-tt-stamp)::@2$3=0::$3=(@-1$4)::$4='(org-tt-current-count)::$5=$4-$3

Название таблицы может быть настроено: 

(setq org-tt-table-name "my-table-name")

org-tracktable не подсчитывает слова

  • в комментариях;
  • в свёрнутых drawers;
  • в заголовках;
  • в тексте под заголовком с тегами nowc или noexport;

Функция org-tt-status покажет

  • количество слов в буфере или в выделенной области;
  • сколько слов сегодня написано;
  • процент от ежедневной цели;

По умолчанию ежедневная цель составляет 300 слов. Изменить это значение можно, настроив переменную org-tt-daily-goal.
